GUI sizing and scaling
-----------------------------------------------------------------------
The GUI size and scaling can be changed using the text file;

FEBiD_CAD_GUI_Info.txt

This file is located in the sub folder;

\EBiD 3D (CAD)\FEBiD 3D CAD (Supporting Files)\GUI Size and Scale

A multiplication factor is used to shrink/grow the interface size.  The font size can also be modified, at least within a certain range.
